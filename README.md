# 📈 Análise de Performance de Receita - E-commerce Olist

Análise de performance de receita e comportamento de vendas utilizando Python e o dataset do E-commerce Olist. Foco em KPIs financeiros e tendências sazonais.
Este projeto apresenta uma análise técnica da performance financeira do ecossistema Olist, transformando dados brutos em insights de negócio. O objetivo é demonstrar competências em manipulação de dados, análise de tendências e comunicação visual.

## 🛠️ Habilidades Técnicas Demonstradas

* **Manipulação de Dados:** Uso de `Pandas` para limpeza, tratamento de tipos (datetime) e filtragem de outliers.
* **Engenharia de Dados (Joins):** Consolidação de múltiplas fontes de dados (pedidos, pagamentos, itens e produtos) em uma arquitetura unificada.
* **Análise de Performance:** Cálculo de KPIs de receita total e quebra por status de pedido.
* **Análise de Tendências:** Identificação de sazonalidade (Black Friday) e crescimento mensal (MoM).
* **Data Quality:** Tratamento de viés de truncamento em dados históricos para garantir a integridade da análise.

## 📊 Principais Insights

1. **Volume de Receita:** Processamento de uma base com faturamento superior a R$ 16 milhões.
2. **Sazonalidade:** Identificado pico histórico em Novembro/2017 (Black Friday).
3. **Integridade:** Realizado o tratamento de dados de Setembro/2018 para evitar conclusões errôneas devido a dados parciais.
